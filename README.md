OrbitalGuard AI is an interactive space situational awareness and orbital debris monitoring platform developed by Peculiar Marcellinus. The project combines web technologies, data visualization, artificial intelligence concepts, and aerospace engineering ideas to demonstrate how future autonomous robotic spacecraft could help solve the growing problem of space debris.
The platform provides an intuitive Mission Control interface that visualizes satellites and orbital debris around Earth while displaying key orbital statistics such as tracked objects, collision alerts, orbital distribution, debris mass, and AI-generated capture priorities. It also includes an interactive 3D Earth visualization, searchable satellite information, and a real-time dashboard inspired by modern aerospace mission control systems.
